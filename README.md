# map-mcp

Drive and perceive an existing **live MapLibre GL map** from an AI agent (over MCP) or a human
CLI — query rendered features, read the viewport, click and read popups, navigate, toggle
layers. A map app opts in with a small JS hook; the agent and the CLI act on the *same* map a
human is looking at, with parity by construction.

> v1 drives *existing* maps (it does not generate maps), MapLibre GL only. Full README lands
> with packaging (U8).
